### Features

#### Simulations
- Create and manage ocean microbiome simulations
- Adjust environmental parameters (temperature, nutrients, light, salinity)
- Step through simulation time periods
- AI-powered predictions for future states
- Reset simulations to initial state
- View simulation history and trends

#### Sensors
- Create sensor zones with geographic coordinates
- Real-time sensor data streaming
- Monitor temperature, pH, salinity, oxygen, and carbon levels
- Simulate environmental events (algae bloom, pollution, storm)
- View current and historical readings

#### Dashboard
- View platform statistics
- Track total simulations
- Monitor active sensors
- Calculate total carbon sequestered
- Analyze ecosystem health metrics
- Track microbe populations

#### Educational
- Learn about ocean microbiomes
- Understand carbon sequestration
- Explore restoration techniques
- Interactive learning modules

## How to Update

When deploying updates:

1. Update version in `package.json` (frontend)
2. Update version in API response (backend)
3. Document changes in this CHANGELOG
4. Tag the release in Git
5. Deploy to Render
